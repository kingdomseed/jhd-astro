# Issue Tracker: Linear

Issues and PRDs for this repo live in Linear.

## Conventions

- Team: `JAD`.
- Project: `jhd-astro` once that Linear project exists.
- Until the project exists, pause before creating website issues instead of
  creating unprojected issues.
- Creating or updating Linear issues, projects, labels, or other external
  records requires explicit user confirmation.
- When a skill says "publish to the issue tracker", draft the Linear issue
  first and ask for confirmation.
- When a skill says "fetch the relevant ticket", read the referenced Linear
  issue if available.
