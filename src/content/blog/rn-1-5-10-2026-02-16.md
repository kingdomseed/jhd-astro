Released on February 16, 2026

Short one this time — mostly cleanup from v1.5.9, plus one new feature that the community's been asking about.

## Highlights

- **Game Services toggle**: You can now opt in to platform achievements (Google Play Games on Android, Game Center on iOS/macOS) from Settings. It's off by default — your call whether you want the achievement tracking.
- **Android link fixes**: "Contact Support" links (website, Discord, documentation, email) now open reliably on all Android devices, including ones where `canLaunchUrl` is blocked by Android 11+ package visibility restrictions.
- **Import improvements**: The conflict preview/restore dialog now handles camelCase names and Map-type list entries properly, which means fewer false conflicts when importing journals between devices.

## Added

- **Game Services**: New "Game Services" toggle in Settings (opt-in) for platform achievements via Google Play Games and Apple Game Center.

## Changed

- Reduced scene action button padding in the Game Log footer to prevent overflow on small layouts.
- Improved import conflict preview to handle camelCase names and Map-type list entries, reducing false conflicts during journal imports.

## Fixed

- **Android**: Fixed "Contact Support" links not opening on some devices.
- **Android**: Prevented Google Play Games sign-in prompts from appearing unless "Game Services" is explicitly enabled.
- **Android**: Improved URL opening reliability even when `canLaunchUrl` is blocked by Android 11+ package visibility.
- Fixed metadata and tag dialogs to close properly, preventing navigation errors after dismissal.
