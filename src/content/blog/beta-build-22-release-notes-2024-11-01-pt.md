Lançado em 1 de Novembro de 2024

Este post faz parte da série Beta → Primeiro Lançamento.

## Melhorias Principais
- Fator Caos agora é um valor universal
- Adicionado relatório de falhas Firebase para rastreamento de estabilidade melhorado
- Arquitetura de banco de dados otimizada para futura migração NoSQL
- Texto de ajuda expandido em todas as telas (via botão de ajuda)

## Rolador de Dados
- Usabilidade melhorada com texto de ajuda mais claro
- Deslize para excluir para rolagens salvas
- Múltiplas correções de bugs
- Testes adicionais em andamento

## Listas de Aventura
- Resolvido bug relacionado a espaço
- Interface redesenhada com três abas:
  - Personagens
  - Tramas
  - Características (rolagem independente)

## Gerenciador de Diário
- Importação de diário com suporte a conversão
- Renomeação de diário
- Exportação básica Markdown
- Corrigido "Não mostrar ao iniciar"
- Melhorias gerais de UI/UX

## Acessibilidade
- Modo canhoto para Tabela de Destino e Verificação de Destino
- Log limitado a 25 entradas com função Limpar

## Gerenciamento de Cena
- Melhorias menores de interface e interação mais suave

## Foco Futuro
- Acessibilidade (rótulos semânticos, modos daltônicos, otimização para leitores de tela)
- Correções de bugs críticos
- Preparação e otimização de lançamento
- Expansão de recursos pós-lançamento
