# Registro de Mudanças — Build de Lançamento Próxima → v1.2.5

## Melhorias do Rolador de Dados
- Corrigido e validado suporte para notações de dados complexas incluindo:
  - Operações Manter/Descartar (ex., `4d6kh3`)
  - Rolagens de Vantagem/Desvantagem (`2d20kh1` / `2d20kl1`)
  - Dados Fudge/Fate (`dF`)
  - Dados Percentuais (`d%`)
  - Rolagens D66
  - Combinações aritméticas complexas
- Melhor manuseio de modificadores em fórmulas de dados
- Adicionado conjunto de testes abrangente para verificar cálculos de rolagem de dados
- Validação aprimorada para todas as notações de dados suportadas
- Funcionalidade Limpar Tudo está mais clara para ambos os botões

## Recursos de Diário
- Modal de confirmação ao excluir entradas de diário
- Feedback tátil em interações de lista em Listas de Aventura
- Melhor gerenciamento de dados e recursos de segurança

## Melhorias Gerais
- Tratamento de erro aprimorado em operações de rolagem de dados
- Melhor validaidação para resultados de rolagem de dados
- Feedback melhorado para ações do usuário
- Texto de ajuda atualizado para clareza
- Nova tela de Suporte com opções de Contatar Suporte
- Tabelas de Significado têm texto copiável
