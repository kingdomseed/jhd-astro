Lançado em 8 de Novembro de 2024

Este post faz parte da série Beta → Primeiro Lançamento.

## Navegação de App Melhorada
- Gaveta de app redesenhada com um visual mais limpo
- Animações suaves para abrir/fechar seções
- Melhor organização de menu com categorias claras

## Melhor Desempenho
- Ajustes de Cena mais responsivos
- Operações de carregar/salvar diário mais rápidas
- Removidos cálculos desnecessários que deixavam as coisas lentas

## Melhorias Visuais
- Escala de texto melhorada em todo o app
- Melhor manuseio do espaço de tela em vários menus

## Correções de Bugs
- Corrigidos problemas de comportamento do botão voltar
- Corrigido bug de listas que impedia o salvamento de dados de lista
- Tratamento de erro mais robusto ao criar novos diários
- Estabilidade melhorada durante ajustes de cena múltiplos

## Nos Bastidores
- Melhorias de confiabilidade em todos os módulos
- Código não utilizado removido
- Melhor organização interna de arquivos e recursos
