Originally previewed on March 25, 2026. This post now reflects the shipped 1.6.0 release line through 1.6.4.

The 1.6 release line is the biggest feature release since the app launched. It adds several Mythic GME 2e systems that players had been asking for, then follows them with the polish needed to make those systems feel steady in real sessions.

The center of the release is the redesigned Game Panel. Meaning Tables, Event Focus, Progress Tracks, Discovery Checks, and Keyed Scenes now have clearer places to live, so the app can support more Mythic play without turning the table into a pile of panels.

## Highlights

- **Progress Tracks**: Guide a Thread or Horror arc toward its ending with phased progress, flashpoints, Plot Armor, and correction controls for when play takes an unexpected turn.
- **Hybrid Play**: Use physical dice across Fate, tables, Event Focus, Adventure Lists, and the Dice Roller while the app still interprets results and keeps the flow moving.
- **Discovery Checks**: Roll a Discovery Check when the story stalls. The app gives a clear result summary and can update an active Progress Track when the result points that way.
- **Keyed Scenes**: Create one-shot or recurring narrative triggers with optional counters so ambushes, omens, and climactic beats can fire at the right time and log themselves to the Adventure Log.
- **Tabbed Game Panel**: Meaning Tables, Event Focus, Progress Tracks, Discovery Checks, and Keyed Scenes now live in a clearer tabbed panel with richer help and cleaner explanations.

## Added In 1.6.0

- Progress Tracks for Threads and Horror arcs, including phases, flashpoints, Plot Armor, and quick correction controls.
- Hybrid Play prompts for players who want to roll physical dice while still using the app to interpret and record results.
- Discovery Checks for pushing stalled adventures forward, with result summaries and optional Progress Track updates.
- Keyed Scenes with one-shot and recurring triggers, optional counters, automatic triggering, badges, and Adventure Log entries.
- A redesigned Game Panel that gives the new tools room without crowding Meaning Tables or Event Focus.
- Cleaner labels, clearer result wording, and markdown-rendered help sheets for the new 1.6 tools.

## Improved Through 1.6.4

- **Expanded Features flow**: Upgrade and renewal screens now stay inside the app with clearer messaging.
- **First launch**: Fresh installs open more cleanly, and release notes show more reliably.
- **iCloud setup**: First-time cloud enable and sync status handling are more dependable.
- **Modal stability**: Several stale-dismiss and transition crashes were fixed across sheets and dialogs.
- **Table translations**: Built-in Meaning Tables and Event Focus tables now support bundled translations.
- **Custom Table editing**: Progressive editing makes Custom Table names, descriptions, entries, and translations easier to manage while keeping work safer when source tables change.
- **Focus Mode and chronological order**: Focus Mode is cleaner on desktop and tablet layouts, and chronological order options make Adventure Logs and scenes easier to read oldest-to-newest.
- **Reliability fixes**: Table imports, table maintenance, spell-check suggestions, saved-formula loading, file picking, and adventure-list note sheets handle more edge cases gracefully.

## What To Know

If you are coming from the 1.5 line, the Game Panel will feel different. The goal is not to make Mythic heavier. The goal is to give the deeper Mythic tools a place where they can be used during play without forcing you to stop and reorganize everything around them.
