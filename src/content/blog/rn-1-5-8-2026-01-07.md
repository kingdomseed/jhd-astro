Released on January 7, 2026

## Highlights

- **60+ New Premium Tables**: Mythic Magazines 46-61 including Star System Creator, Society Crafter, Spell Crafter, Fantasy World Crafter, City Crafter, and more.
- **Streamlined Fate Chart**: Removed the large odds display card. Target numbers now appear at the top of the adventure log for a cleaner look.
- **Cleaner Roll Fate Button**: Updated visuals for better readability across themes and screen sizes.
- **Button Theming**: Improved theming and contrast for key action buttons.
- **Reorganized Settings**: Game Behavior settings cleaned up and reorganized for easier navigation.
- **Built-in Documentation**: Head to the Support screen to find docs and tutorials.
- **Prepared Adventure Mode**: Scene adjustments hidden by default, no altered/interrupt scenes—just Expected scenes as Mythic 2e recommends for pre-written adventures.
- **Customizable Layout**: Long-press any panel on tablet or desktop to move, swap, replace, or hide it. Arrange Meaning Tables, Game Log, Fate Actions, Lists, and Scenes however you like.
- **Undo**: Bring back deleted log entries, scenes, and custom tables. Cmd/Ctrl+Z on desktop, shake to undo on mobile.
- **Adventure Lists**: Auto-duplicate entries (up to 3 copies), auto-condense when all 5 sections are full.
- **Privacy & Stability**: Clearer analytics privacy explanation, a direct link to the Privacy Policy, and smoother foldable/tablet layouts.

## Added

- 64 new premium tables from Mythic Magazines 46-61:
  - **MM46** (Star System Creator): Star System Exploration Focus
  - **MM49** (Society Crafter): The Beginning Focus, Productivity Focus, Flourish Focus
  - **MM54** (Down Time Adventures): Quest Focus, The Quest, Challenge Focus, The Challenge
  - **MM56** (Detail Checks): Detail Focus
  - **MM57** (Expanding Events & Chases): Plot Points, Foot Chase Mayhem, Vehicle Chase Mayhem
  - **MM58** (Creating Dialogue & Moonlight Radio): Conversation Focus, Conversation Words, Hello Focus, Caller's Voice, Revelation, Scheduled Segments
  - **MM59** (Spell Crafter & Cyber Dystopia): Spell Type, Spell Requirements, Cyber Dystopia World Focus, Cyber Dystopia World, Cybernetic Character Focus, Cybernetic Character, Virtual World Elements, Mission Seeds: Employer, Mission Seeds: Objective, plus 26 Spell Effects tables
  - **MM60** (Fantasy World Crafter): Fantasy Sub-Genre, Central Mythology Focus, Central Society Focus, Magic Focus, Monster Focus, Name Sounds, Threats & Tension Focus
  - **MM61** (City Crafter): City Identity Focus, District Focus, City Features
- New "World Building" category for world setting, campaign framework, and city building tables.
- Built-in Help Center: Head to the Support screen to find searchable documentation and tutorials organized by category.
- Settings reorganized: Game Behavior settings cleaned up for easier navigation.
- Customizable Layout: Long-press any panel on tablet or desktop to swap, replace, hide, or fill empty slots. Choose from Meaning Tables, Game Log, Fate Actions, Lists, and Scenes.
- Fourth panel slot on large screens (iPad Pro 12.9", 1080p+ desktop monitors).
- Scenes panel has a draggable divider between scene list and detail views.
- Undo support (Cmd+Z / Ctrl+Z on desktop, shake on mobile) for deleted log entries, scenes, custom tables, and "Clear All" on adventure lists. Remembers up to 20 actions. Clears when switching journals.
- Game Log: Swipe-to-delete on mobile, delete button on desktop (two-tap for safety).
- Game Log: Question now appears at the top of Fate roll results.
- Random Events button is now optional (toggle in Game Behavior settings). Automatic random events during play still work as normal.
- Game Tables Panel: Event Focus and Meaning Tables are now combined in a unified slide-out panel.
- Meaning Tables: Edit built-in category names, icons, and colors.
- Meaning Tables: Long-press any table to see full info and all entries (Metadata Sheet).
- Meaning Tables: Tag filters now show as chips below the search bar.
- Meaning Tables: Results panel grows to fit longer content (no longer capped at 25%).
- Meaning Tables: Favorites heart icon on desktop for quick access.
- Meaning Tables (Desktop): Hover tooltip (3-second delay) showing full table name and source.
- Meaning Tables: Table entries that say "Roll on X table" now automatically link to and roll on the referenced table.
- Adventure Lists: Auto-duplicate entries up to 3 copies by default (configurable in List Settings).
- Adventure Lists: Auto-condense lists when all 5 sections are full.
- Prepared Adventure Mode: Scene adjustments hidden by default, altered/interrupt scenes removed. Roll <= Chaos Factor gives Expected Scene + Random Event; roll > Chaos Factor gives Expected Scene only—following Mythic 2e rules for pre-written adventures.
- Spell check: Red underlines on macOS, autocorrect on iOS and Android.
- Smarter capitalization: sentences for questions/summaries/notes, words for scene titles.
- Help: Detailed Scene Check and Scene Adjustment rules with rich Markdown formatting.
- Help: Improved help screens on mobile with better formatting.
- Localized tooltips for Scene Check, Scene Adjustment, and Random Event buttons.
- Settings: "Update Table Metadata" button to refresh built-in tables without overwriting custom edits.
- Event Focus: Right-click or long-press any Event Focus table to set it as the active/default table.
- Journal Import: If the imported journal name already exists, you can now rename it (default; suggested “copy”, “copy 1”, etc.) or overwrite the existing journal.

## Changed

- Fate Chart: Removed the large FateDisplay card. Target numbers now shown at top of adventure log.
- Fate roll results display Question at the top, then Result, then Roll.
- Roll Fate button visuals refreshed for a more readable look across themes and screen sizes.
- Location Meaning Table moved to "Core Tables" category to match the rulebook.
- Custom Tables: "Edit" now opens the full editor directly (removed "Advanced Editor" option).
- Adventure Lists: Threads now appear before Characters to match book order.
- Game Log scrolling works while Meaning Tables panel is open.
- "Show Timestamps" setting moved from Accessibility to Log Settings.

## Fixed

- **Critical**: Fate Chart odds no longer reset after rolling on meaning tables, adding scenes, or other journal saves.
- Opening the Privacy Policy and other external links is more reliable (and failures are handled gracefully).
- Premium tables no longer appear locked for users with valid purchase history.
- "Random Event Focus" and "Prepared Event Focus" tables correctly unlock for all users.
- Desktop: Search, favorites, and tag filtering work correctly on meaning tables.
- Desktop: Tags appear properly in the filter panel.
- Adventure Lists: Active sections count persists correctly between sessions.
- Adventure Lists: "Choose" shows roll position when empty slot is rolled.
- Adventure Lists: Duplicate counter excludes "Choose" placeholders.
- Scenes: Pinned scenes reliably move to the top in correct order.
- Scenes: Fixed crash from duplicate timestamps in scene lists.
- iOS: Fixed unresponsive back button on paywall screen.
- iCloud sync crash fixed (malformed filenames).
- Tablet/desktop button colors display correctly.
- Meaning Tables: Favorites filter toggle visible in dark mode.
- ROLL FATE button text shadow no longer appears grey in light mode.
