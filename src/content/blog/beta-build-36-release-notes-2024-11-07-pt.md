Lançado em 7 de Novembro de 2024

Este post faz parte da série Beta → Primeiro Lançamento.

Build 36 será postada amanhã de manhã.

## Acessibilidade & Usabilidade
- Melhor suporte para tamanhos de texto maiores
- Legibilidade melhorada em todas as telas
- Botões e texto mais fáceis de ler em todos os tamanhos
- Conteúdo permanece visível quando o tamanho do texto aumenta
- Tamanhos de texto mínimo e máximo ajustados para legibilidade

## Melhorias de Interface
- Tela de Tabelas de Significado atualizada para melhor usabilidade
- Rolador de Dados se adapta melhor a diferentes tamanhos de tela
- Layouts de botão mais consistentes
- Roda de Probabilidades é mais legível em tamanhos de texto maiores
- Espaçamento aumentado para interação mais fácil

## Melhorias de Confiabilidade
- Salvamento de diário mais confiável
- Sistema de backup de diário melhorado
- Melhor recuperação de erro ao salvar
- Tentativas automáticas se o salvamento falhar
- Melhorias gerais de estabilidade

## Outras Melhorias
- Estilo de botão mais consistente
- Mensagens de erro mais claras e úteis
- Melhor feedback visual em interações de botão
- Ajustes de layout para usar melhor o espaço de tela disponível
- Suporte melhorado para uma gama mais ampla de tamanhos de dispositivo

Essas atualizações focam em tornar o app mais acessível, confiável e fácil de usar para todos, com atenção especial a usuários que dependem de tamanhos de texto maiores ou recursos de acessibilidade. Um agradecimento especial à comunidade pelo feedback nesta área!
