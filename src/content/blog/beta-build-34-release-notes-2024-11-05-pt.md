Lançado em 5 de Novembro de 2024

Este post faz parte da série Beta → Primeiro Lançamento.

Uma atualização está saindo agora para iOS e Android. Amazon a seguir amanhã.

## Melhorias
- Aprimoramentos de visibilidade de texto em todo o app
- Correção para sistema de geração de Evento Aleatório
- Conformidade de privacidade e controles de acesso a dados aprimorados
- Designs de ícone de app atualizados
- Novas capturas de tela e conteúdo de listagem de loja

## Versão — Lançamento Atual

### Melhorias Principais

#### Acessibilidade
- Rótulos semânticos revisados para suporte aprimorado a leitores de tela
- Estrutura de navegação reorganizada para melhor acessibilidade
- Suporte VoiceOver aprimorado em todo o app

#### Estabilidade & Desempenho
- Testes de estresse extensivos do sistema de salvar/carregar
- Múltiplas correções de bugs para evitar falhas durante o jogo
- Confiabilidade de persistência de dados melhorada

#### Interface do Usuário
- Interface de lançador atualizada com refinamentos visuais
- Convertidas várias caixas de texto para interruptores para melhor usabilidade
- Consistência visual aprimorada entre telas

## Atualizações de Recursos

### Rolador de Dados
- Adicionado suporte para múltiplos formatos de notação de dados
- Aceita tanto espaços quanto vírgulas entre tipos de dados  
  Exemplos: `2d6 1d8` ou `2d6,1d8`

## Melhorias Técnicas
- Tratamento de erro robusto para operações de salvamento
- Validação de dados aprimorada para persistência do estado do jogo
- Desempenho otimizado durante sessões de jogo estendidas

## Problemas Conhecidos
- Algumas preocupações de visibilidade de texto ainda sendo abordadas
- Refinamentos menores de UI pendentes
- Eventos Aleatórios podem não disparar corretamente em alguns casos extremos

## Notas
- Por favor, relate problemas e feedback através do canal de suporte do app
- Faça backup dos seus jogos salvos regularmente enquanto melhorias no sistema de salvamento continuam
